# Project Brief

## Obiettivo
Realizzare una web app per "La Cucina Italiana" che consenta di consultare ricette e acquistare ricette complete (insieme di ingredienti nelle quantita' corrette) per un numero di persone scelto.

## Scope
- Database: creare e popolare con dati realistici (ricette, utenti, ingredienti, vini).
- Web app: login, lista ricette, filtri per genere, carrello, checkout, logout.
- Frontend + backend separati (nessun framework obbligatorio).

## Requisiti Chiave
- Calcolo corretto del costo per persona e per ricetta.
- Utenti non autenticati: solo consultazione ricette e costi; niente carrello/acquisti.
- Persistenza acquisti e svuotamento carrello al checkout.

## Vincoli
- Interfaccia semplice ma chiara e funzionale.
- Architettura spiegabile a lezione.
